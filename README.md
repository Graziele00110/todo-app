# To-Do App

Aplicação web desenvolvida para gerenciamento de tarefas e organização pessoal, com interface simples, intuitiva e responsiva.

---

## 🚀 Funcionalidades

- Adição de tarefas
- Remoção de tarefas
- Organização de atividades
- Interface responsiva
- Gerenciamento de produtividade

---

## 💻 Tecnologias

- HTML
- CSS
- JavaScript
